es6学习笔记06/函数扩展

es6扩展了一些Function数据类型api，方便进行数组操作。

函数参数的默认值

es6之前只能通过或给参数赋值默认值，比较麻烦。es6允许为函数的参数直接设置默认值，写在参数定义后面就可以。

{
  function test(x, y = 'world'){
    console.log('默认值',x,y);
  }
  test('hello');  //hello world
  test('hello','kill');  //hello kill
}

  • 作用域：一旦设置了参数的默认值，函数进行声明初始化时，参数会形成一个单独的作用域（context）。等到初始化结束，这个作用域就会消失。这种语法行为，在不设置参数默认值时，是不会出现的。

    {
  let x='test';
  function test2(x,y=x){
    console.log('作用域',x,y);
  }
  test2('kill');  //kill kill
}

{
  let x='test';
  function test2(x,y=x){
    console.log('作用域',x,y);
  }
  test2();  //kill kill
}

es6学习笔记07/对象扩展

es6扩展了一些Object数据类型api，方便进行数组操作。

简洁表示法

ES6 允许直接写入变量和函数，作为对象的属性和方法。这样的书写更加简洁。

{
  // 简洁表示法
  let o=1;
  let k=2;
  let es5={
    o:o,
    k:k
  };
  let es6={
    o,
    k
  };
  console.log(es5,es6);

  let es5_method={
    hello:function(){
      console.log('hello');
    }
  };
  let es6_method={
    hello(){
      console.log('hello');
    }
  };
  console.log(es5_method.hello(),es6_method.hello());
}

ES6允许在对象中直接写变量，属性名为变量名，属性值为变量值，且属性名为字符串类型。

属性表达式

JavaScript语言定义对象的属性，有两种方法。

  • 方法一是直接用标识符作为属性名
  • 方法二是用表达式作为属性名，这时要将表达式放在方括号之内。 
// 属性表达式
let a='b';
let es5_obj={  // {a: "c", b: "c"}
a:'c',
b:'c'
};

let es6_obj={  // {b: "c"}
[a]:'c'
}

console.log(es5_obj,es6_obj);

值得注意的是，属性名表达式如果是一个对象，默认情况下会自动将对象转换为字符串object Object

const keyA = {a: 1};
const keyB = {b: 2};

const myObject = {
  [keyA]: 'valueA',
  [keyB]: 'valueB'
};

myObject // Object {[object Object]: "valueB"}

所以会覆盖[keyA]属性。

新增api

  • Object.is()

    console.log('字符串',Object.is('abc','abc'),'abc'==='abc');

Object.is()用于判断两个值是否严格相等，与es5的===功能相似。不同的是该方法修正了严格相等的两处不适合的地方，它们分别是+0不等于-0，NaN等于NaN。

+0 === -0 //true
NaN === NaN // false

Object.is(+0, -0) // false
Object.is(NaN, NaN) // true
  • Object.assign()

Object.assign方法用于对象的合并，将源对象（source）的所有可枚举属性，复制到目标对象（target）。

console.log('拷贝',Object.assign({a:'a'},{b:'b'}));  //{a: "a", b: "b"}

  • Object.keys()，Object.values()，Object.entries()
    ES5 引入了Object.keys方法，返回一个数组，成员是参数对象自身的（不含继承的）所有可遍历（enumerable）属性的键名。
    Object.values方法返回一个数组，成员是参数对象自身的（不含继承的）所有可遍历（enumerable）属性的键值。
    Object.entries方法返回一个数组，成员是参数对象自身的（不含继承的）所有可遍历（enumerable）属性的键值对数组。
    Object.entries()方法的另一个用处是，将对象转为真正的Map结构。

    var obj = { foo: 'bar', baz: 42 };
var map = new Map(Object.entries(obj));
map // Map { foo: "bar", baz: 42 }

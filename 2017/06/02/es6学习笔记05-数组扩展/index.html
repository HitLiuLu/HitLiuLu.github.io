es6学习笔记05/数组扩展

es6扩展了一些Array数据类型api，方便进行数组操作。

Array.of()

Array.of()方法用于将一组值转换成数组。

{
    let arr = Array.of(3,4,7,9,11);
    console.log('arr=',arr);  //arr=[3, 4, 7, 9, 11]

    let empty = Array.of();
    console.log('empty',empty);  //empty []
}

Array.of() 和 Array 构造函数之间的区别在于处理整数参数：Array.of(7) 创建一个具有单个元素 7 的数组，而 Array(7) 创建一个包含 7 个 undefined 元素的数组。

Array.from()

Array.from() 方法从一个类似数组或可迭代的对象中创建一个新的数组实例。

{
    let p = document.querySelectorAll('p');
    let pArr = Array.from(p);
    pArr.forEach(item=> {
        console.log(item.textContent);
    });

    console.log(Array.from([1,3,5],item=>item*2))
}

可以看到Array.from()方法将页面中p标签组成的nodeList转换成数组。同时，该方法可以传递两个参数，第二个参数可以接受一个方法，将第一个的数组或类集合数组进行映射，返回结果。

数组实例的fill()

fill方法使用给定值，填充一个数组。

{
  console.log('fill-7',[1,'a',undefined].fill(7));
  console.log('fill,pos',['a','b','c'].fill(7,1,3));
}


fill方法可以用于空数组的初始化，如果数组不为空，数组中的值将被全部抹去。fill方法还可以接受第二个和第三个参数，用于指定填充的起始和截止位置。

数组实例的entries(),keys()和values()

ES6提供了三个新方法－－entries(),keys()和values()－－用于遍历数组。可以用for…of循环进行遍历，区别是keys()是对键名的遍历、values()是对键值的遍历，entries()是对键值对的遍历。

{
  for(let index of ['1','c','ks'].keys()){
    console.log('keys',index);
  }

  // for(let value of ['1','c','ks'].values()){
  //   console.log('values',value);
  // } //chrome 尚未实现

  for(let [index,value] of ['1','c','ks'].entries()){
    console.log('values',index,value);
  }
}

copyWith()

copyWith()方法在当前数组内部，将指定位置的成员复制到其他位置（会覆盖原成员），然后返回但前数组。

{
  console.log([1,2,3,4,5].copyWithin(0,3,4));
  // [4,2,3,4,5]
}

find()和findIndex()

find方法用于寻找符合条件的第一个数组成员，找到第一个并返回该成员，若没有找到符合条件的成员返回undefined。
findIndex方法用于返回符合条件的第一个数组成员的位置，如果所有成员都不符合条件则返回－1。

{
  console.log([1,2,3,4,5,6].find(item=> item>3)); // 4
  console.log([1,2,3,4,5,6].findIndex(item=> item>3));  // 3
}

includes()

该方法返回一个布尔值，表示某个数组是否包含给定的值。

{
  console.log('number',[1,2,NaN].includes(1));  // true
  console.log('number',[1,2,NaN].includes(NaN)); // true es5做不到
}

es6学习笔记03/字符串扩展

es6新增了一些api以简化字符串的处理方法。下面对一些较为常用的api简单记录一下。

判断字符串包含、起始和结束

对字符串中是否包含子字符串，是否以某个子字符串起始和结束，在es6中我们可以分别使用includes(),startsWith(),endsWith()方法。

{
    //字符串包含某字符串，是否以某字符串开始或结束
    let str = "string"
    console.log('includes',str.includes("ring"))
    console.log('start',str.startsWith("str"))
    console.log('end',str.endsWith("ng"))
}
//结果均为true

重复字符串

多次重复一个字符串可以使用repeat(n)方法。n表示要重复的次数。

{
    let str = "abc";
    console.log(str.repeat(2));
}
//abcabc

模版字符串

使用模版字符串可以减少一些复杂的字符串拼接。

{    
    //模板字符串
    let name = "list";
    let info = "hello world";
    let m = `i am ${name}, ${info}`;
    console.log(m);
}

其中数据部分用${ }的形式包在大括号里。

补白效果

这一部分实用性效果很好，不过es6中尚未实现，应该属于es7的特性。

{
    console.log('1',padStart(2,'0')); 
     console.log('1',padEnd(2,'0'));
}
//01 10

补白的效果可以应用在标准化日期时在月份和日期前补0。

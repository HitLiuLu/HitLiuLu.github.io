es6学习笔记09/Set与Map数据结构

es6提供了新的数据结构Set, Map,下面一一分析。

Set

Set数据结构类似于数组，但成员值是唯一的，没有重复值。

{
    let list = new Set();
    list.add(5);
    list.add(7);

    console.log('size', list.size);
}

Set可以通过构造函数生成Set数据结构。然后可以通过add()方法向Set结构加入成员。

Set函数可以接受一个数组或类似数组的对象作为参数初始化。

{
    let arr = [1,2,3,4,5];
    let list = new Set(arr);

    console.log(list);  // Set(5) {1, 2, 3, 4, 5}
}

Set的去重功能

Set数据的一个重要应用场景就是数组去重。

{
    //功能：去重
    let list = new Set();
    list.add(1);
    list.add(2);
    list.add(1);

    console.log('list',list);  //list Set(2) {1, 2}

    let arr=[1,2,3,1,2];
    let list2 = new Set(arr);

    console.log('unique',list2);  //unique Set(3) {1, 2, 3} 

}

值得注意的是，Set内部判断两个值是否相等使用的算法类似于严格全等，不同之处是NaN等于自身，而严格全等认为NaN不等于自身。

let set=New Set();
let a=NaN;
let b=NaN;
set.add(a);
set.add(b);
set;  // Set {NaN}

Set的属性和方法

let arr=['add','delete','clear','has'];
let list=new Set(arr);

console.log('has',list.has('add'));  // has true
console.log('delete',list.delete('add'),list, list.size);  
// delete true  Set(3) {"delete", "clear", "has"} 3

list.clear();  
console.log('list',list);  // Set(0) {}

上面我们用到了add()方法，该方法用于给Set实例添加成员。has()方法返回一个布尔值，用于表示该值是否是Set成员，delete()方法删除某个值，返回一个布尔值，表示是否删除成功。clear()方法用于清空所有成员，没有返回值。size属性用于返回Set实例的成员个数。

Set遍历操作

Set结构实例可以有四个遍历方法，可用于遍历成员。

  • keys():返回键名的遍历器
  • values():返回键值的遍历器
  • entries():返回键值对的遍历器
  • forEach():使用回调函数遍历每个成员

由于Set结构没有键名，只有键值，所以keys()与values()方法的行为完全一致。entries()返回的键名和键值构成的数组。

let arr=['add','delete','clear','has'];
let list=new Set(arr);

for (let key of list.keys()) {
    console.log('key',key);
}
//add
//delete
//clear
//has
for (let value of list.values()) {
    console.log('value',value);
}
//add
//delete
//clear
//has
for (let [key,value] of list.entries()) {
    console.log('entries',key,value);
}
//add add
//delete delete
//clear clear
//has has

Set结构实例的forEach方法用于对每个成员执行某种操作，没有返回值。

list.forEach(item => {
    console.log(item);
})
//add
//delete
//clear
//has

WeakSet

WeakSet结构与Set结构类似，也是不重复的值的集合。但它的成员只能是对象，不能是其它数据类型值。

let weakList = new WeakSet();

let arg={};

weakList.add(arg);

console.log('weakList',weakList);  //weakList WeakSet {Object {}}

同样的，与Set相比，没有clear()方法，size属性而且不能遍历。

weakList.add(2);  //TypeError: Invalid value used in weak set

Map

es6提出的Map结构，类似于对象，也是一个键值对的集合，但是它的键不仅限于字符串，各种类型的值都可以当作键。Object结构提供了“字符串－值”的对应，Map结构提供了“值－值”的对应，是一种更加完善的Hash结构。对于键值对这种数据结构，Map比Object合适。

let map = new Map();
let arr=['123'];

map.set(arr,456);

console.log('map',map);  //map Map(1) {["123"] => 456} 456 

map.get(arr);  // 456

Map的属性和方法

let map = new Map([['a',123],['b',456]]);
console.log('map args',map);
//map args Map(2) {"a" => 123, "b" => 456}

console.log('size',map.size);
//2

console.log('delete',map.delete('a'),map);
//delete true Map(1) {"b" => 456}

console.log('clear',map.clear(),map);
//clear undefined Map(0) {}

作为构造函数，Map可以接受一个数组作为参数。事实上，不仅仅是数组，任何具有Iterator接口的数据结构都可以当作Map构造函数的参数。也就是说Set, Map都可以用来生成新的Map。

  • set(key, value)
    set方法设置键名key对应的value，然后返回整个Map结构。如果key已经有值，则键值会被更新，否则就新生成该值。

    const m=new Map();

m.set('abc',13);
m.set(13,'abc');
m.set(undefined,'aab');
m  // Map(3) {"abc" => 13, 13 => "abc", undefined => "aab"}

set方法返回的是当前的Map对象，可以采用链式结构实例化Map。

let map=new Map()
    .set(1,'a')
    .set(2,'b')
    .set(3,'c')

map  // Map(3) {1 => "a", 2 => "b", 3 => "c"}

  • get(key)
    get方法读取key对应的键值，如果找不到，返回undefined。

    let newMap=new Map();
newMap.set(newMap, "newMap");
newMap.get(newMap);  // "newMap"

WeakMap

{
    let weakmap=new WeakMap();

    let o={};
    weakmap.set(o,123);
    console.log(weakmap.get(o));  // 123
}

Map-WeakMap可以对标Set-WeakSet。

更具体内容请参考《ECMAScript 6 入门 阮一峰》

es6学习笔记08/Symbol

ES6引入了一种新的原始数据类型Symbol，表示独一无二的值。它是JavaScript语言的第七种数据类型。七种类型包括5种简单数据类型（也称为基本数据类型）：Undefined,Null,Boolean,Number和String。还有一种复杂数据类型Object,Object本质上是由一组无序的键值对组成。es6又引入一种数据类型Symbol。

Symbol的声明

Symbol的值通过函数Symbol()生成。

let a1=Symbol();
let a2=Symbol();
console.log(a1===a2);
//false

typeof a1;
//symbol

可以看到Symbol类型声明的值是独一无二的，通过typeof运算符我们看到变量a1是Symbol数据类型。Symbol数据类型与其它数据类型不同的地方是它不可以使用new命令，使用new命令会报错。Symbol是一个原始类型值，它不是对象，所以不能添加属性。

Symbol函数可以接受一个字符串作为参数，表示对Symbol实例的描述，主要用于区分。

let a3=Symbol('a3');
let a4=Symbol('a4');

a3;  // Symbol(a3)
a4;  // Symbol(a4)

a3.toString();  // "Symbol(a3)"
a4.toString();  // "Symbol(a4)"

可以看到，有了参数以后，就如同给它们加了描述，输出的时候就能分清到底哪个是哪个。因为函数的参数只是为了描述和区分，因此，就算相同的参数，它们也是不同的。

let a5=Symbol('a5');
let a6=Symbol('a6');

a5===a6  // false

Symbol值不能与其它类型的值进行运算，会报错，但可以转为字符串和布尔值，但不能转为数字。

let sym=Symbol('My Symbol');

String(sym);  // "Symbol(My Symbol)"

Boolean(sym);  // true

Number(sym);  // TypeError

Symbol.for()

有时我们希望重新使用同一个Symbol值，就需要用到Symbol.for()方法。它接受一个字符串作参数，然后搜索有没有以该参数作为名称的Symbol值。如果有就返回这个Symbol值，否则就新建并返回一个以该字符串为名称的Symbol值。

var s1=Symbol.for('foo');
var s2=Symbol.for('foo');

s1===s2  // true
typeof s1  // "symbol"

s1和s2都是Symbol值，它们都是同样参数的Symbol.for()方法生成的，所以是同一个值。

Symbol.for()与Symbol()不同之处在于前者会被登记在全局环境中供搜索，后者不会。前者在调用前会先检查检查给定的key值是否已经存在，如果不存在才会新建一个Symbol类型的值，而后者每次都会新建一个独一无二的Symbol类型值，也就是说每次返回的值都是不同的。

作对象的属性名

由于每一个Symbol值是不同的，这意味着Symbol值可以作为标识符，用于对象名的属性名，可以保证不会出现同名属性。这对于一个对象由多个模块构成的情况非常有用，可以防止某一个键被不小心改写或覆盖。

let a1=Symbol.for('abc');
let obj={
  [a1]:'123',
  'abc':345,
  'c':456
};
console.log('obj',obj);  // obj {abc: 345, c: 456, Symbol(abc): "123"}

需要注意的是，Symbol值作为对象属性名时，不能用点运算符。

obj.a1='replace';
obj['a1'];  // 'replace'
obj;  // {abc: 345, c: 456, a1: "replace", Symbol(abc): "123"}
obj[a1];  // Symbol(abc): "123"

属性名的遍历

Symbol作为属性名，该属性不会出现在for…in, for…of循环中，也不会被Object.keys(), Object.getOwnPropertyNames(), JSON.stringify()返回。但是，他也不是私有属性，获取对象的Symbol属性名可以通过Object.getOwnPropertySymbols()。
该方法返回一个数组，成员是当前对象所有用作属性名的Symbol值。

let objectSymbols=Object.getOwnPropertySymbols(obj);
objectSymbols;  // [Symbol(abc)]

Object.getOwnPropertySymbols方法与for…in循环，Object.getOwnPropertyNames对比

var obj={};

var foo=Symbol("foo");

Object.defineProperty(obj,foo,{
    value: "foobar"
});

for (var i in obj) {
    console.log(i);  // 无输出
}

Object.getOwnPropertyNames(obj);  // []

Object.getOwnPropertySymbols(obj)  //[Symbol(foo)]

更具体内容请参考《ECMAScript 6 入门 阮一峰》

正则表达式－－ES6新增

es6中为正则表达式新增了一些特性，下面找几个常用的记录下来。

构造函数

ES5和ES6允许在创建正则表达式的时候使用字面量和构造函数的方法，相同的是，它们中的构造函数方法都可以接收两个参数，不同的是es5接收的第一个参数只可以是要匹配的字符串，而es6接收的第一个参数可以是一个正则表达式。例如：

{
    //es5
    let regex = new RegExp('xyz','i')
    let regex2 = new RegExp(/xyz/i)

    console.log(regex.test('xyz123'),regex2.test('xyz123'))

    //es6
    let regex3 = new RegExp(/xyz/ig,'i') //后面的修饰符会覆盖前面的正则表达式所用到的修饰符
    console.log(regex3.flags) //输出i
}

y修饰符

我们熟悉g修饰符，它表示全局匹配。y修饰符也是表示全局匹配，不同点是，g修饰符从上一次的匹配位置接着往下找，直到找到符合条件的匹配结果。而y修饰符强调必须是从上一次匹配位置的紧跟着的下一个开始。
看一个例子：
{
let s = ‘bbb_bb_b’
let a1 = /b+/g
let a2 = /b+/y

    console.log('one',a1.exec(s),a2.exec(s))
    console.log('two',a1.exec(s),a2.exec(s))

    console.log(a1.sticky,a2.sticky)
}

es6中新增的sticky属性可以判断正则表达式是否开启了粘连模式————y修饰符的这种模式。

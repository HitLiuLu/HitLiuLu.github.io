Markdown常用语法

星号与下划线都可以，单是斜体，双是粗体，符号可跨行，符号可加空格

4. 列表

无序列表使用*+或是-作为列表标记

*   Blue

+   Red

也等同于

-   Red

-   Blue

  • Blue

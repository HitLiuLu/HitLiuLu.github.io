es6学习笔记01/let、const

ES6新增了let命令，用来声明变量。它的用法类似于var，但是所声明的变量，只在let命令所在的代码块内有效。
const声明一个只读的常量。一旦声明，常量的值就不能改变。

块级作用域

ES6在ES5作用域基础上新增了块级作用域，本来ES5只有全局作用域和函数作用域。
块级作用域我们可以这么简单理解：一段代码是由一个大括号包围的区域就是块级作用域。而let声明的变量只在块级作用域内有效。

let命令

function test() {
    for (let i = 1; i < 3; i++) {
        console.log(i);
    }
    console.log(i);
}

test();

打开控制台，我们可以发现输出了1、2，接着报了一个引用错误。由于es6强制开启严格模式，所以引用错误是由于变量i没有声明导致的。所以let与var的不同点就在于let定义的变量只在块级作用域内有效。脱离了块级作用域，变量i是未声明的。
值得注意的是，let命令不可以在同一块作用域内重复定义变量。
例如： 

function test() {
    let a = 1;
    let a = 2;
}

test();

代码报错，错误类型是不可以重复定义变量。

const命令

const声明一个read-only的常量。一旦声明，常量的值就不能改变。

例如：

function last() {
    const PI = 3.1415926;
    PI = 8;   //read-only
    console.log(PI);
}

last();

代码报错，显示PI属性read-only，不可改变。

function last() {
    const PI;
    PI = 8;   //声明的时候必须赋值
    console.log(PI);
}

last();

代码报错，声明的时候必须赋值。

更具体内容请参考《ECMAScript 6 入门 阮一峰》
